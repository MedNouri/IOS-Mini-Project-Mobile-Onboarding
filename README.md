# IOS-Mini-Project-Mobile-Onboarding
Mini Project Mobile Onboarding FOR  IOS

## Features
-  UICollection View Diffable DataSource .
-  UICollection View Compositional Layout 
-  CABasicAnimation
-  Support for dark mode


## Requirements
- Xcode 11.4

## Getting Started

- Clone or download
- Build and Run and have fun
